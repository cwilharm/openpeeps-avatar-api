Quick Start

1. Include the script

// Add to your HTML <script src="avatar-builder.js"></script>

2. Use the component

// Add the Web Component <avatar-builder api-url="http://localhost:8000"></avatar-builder>

3. Listen to events (optional)

const builder = document.querySelector('avatar-builder'); builder.addEventListener('avatar-generated', (event) => { console.log('Key:', event.detail.key); console.log('SVG:', event.detail.svg); });

API Endpoints

GET
/options

Returns all available component options for each category

// No parameters required Response: { categories: { head: [...], face: [...], ... } }
POST
/avatar/generate

Generate an avatar with specific component selection

// Body Parameters (JSON) { "head": 0, // Index of head style "face": 0, // Index of face expression "body": 0, // Index of body/clothing "facial_hair": 0, // Index of facial hair "accessories": 0 // Index of accessories } // Response { "key": "0e64267e", "svg": "<svg...>" }
GET
/avatar/{key}

Retrieve a previously generated avatar by its key

// Path Parameter key: "0e64267e" // The avatar key from generate // Response { "key": "0e64267e", "svg": "<svg...>" }
GET
/avatar/random

Generate a random avatar with random component selection

// No parameters required // Response { "key": "7ce711ed", "svg": "<svg...>" }
